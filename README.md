# 42_ft_ssl_md5

args using bitwise impl: https://youtu.be/6hnLMnid1M0

1. Bitwise AND (`&`):
```
    01101100
&   10101010
------------
    00101000
```

2. Bitwise OR (`|`):
```
    01101100
|   10101010
------------
    11101110
```

3. Bitwise XOR (`^`):
```
    01101100
^   10101010
------------
    11000110
```

4. Bitwise complement (`~`):
```
~   01101100
------------
    10010011
```

5. Bitwise left shift (`<<`):
```
    01101100
<<  3
------------
    01100000
```

6. Bitwise right shift (`>>`):
```
    01101100
>>  3
------------
    00001101
```
